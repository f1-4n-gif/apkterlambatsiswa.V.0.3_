Cara Instal di HP Android (PWA)

Aplikasi ini dirancang khusus agar bisa dipasang di Layar Utama HP Anda layaknya aplikasi asli tanpa memakan banyak memori.

  1. Buka link aplikasi ini di aplikasi browser Google Chrome pada HP Android Anda.
  2. Tekan tombol "Titik Tiga" (Menu) di pojok kanan atas Google Chrome.
  3. Pilih opsi "Tambahkan ke Layar Utama" (Add to Home Screen) atau "Instal Aplikasi".
  4. Setelah berhasil, Anda bisa menutup Chrome dan mencari ikon "E-Terlambat" langsung di layar utama HP Anda!

1. Backup & Restore Data

  • Karena data ini tersimpan di memori HP/Browser secara *Offline*, gunakan tombol Backup Data di menu untuk menyimpan seluruh data ke dalam satu file.
  • Lakukan Backup secara berkala. Jika Anda tidak sengaja membersihkan riwayat browser HP, gunakan Restore Data dan masukkan file backup tadi.

2. Mengelola Data Siswa

  • Pastikan Anda mengisi data siswa terlebih dahulu sebelum menggunakan fitur scan.
  • Gunakan fitur Input Massal untuk memasukkan ratusan data sekaligus dari File Excel (.xlsx).
  • Unduh Format Excel, isi data (NISN, Nama Siswa, Kelas, No. HP Ortu), simpan, lalu unggah ke aplikasi.

3. Scan & Kirim WA (Kamera HP Android)

  • Di halaman Scan, tekan tombol Gunakan Kamera HP. Saat ditanya izin oleh Android, Wajib Pilih 'Izinkan' (Allow).
  • Tukar Kamera: Gunakan tombol Tukar Kamera di bawah kotak layar untuk beralih antara kamera Depan (Selfie) dan Belakang.
  • Pengaturan WA Otomatis: Jika Anda mengatur API Fonnte di menu Pengaturan WA, pesan terkirim otomatis di latar belakang.
  • Jika memilih mode Manual, gunakan sakelar Sistem Antrean WA agar Anda dapat mem-batch pengiriman setiap 40 siswa supaya proses scan kamera Anda tidak terpotong-potong.
